Features

为 Kimi Code 而生

轻量驻留菜单栏，不占 Dock、不打断心流，需要时一眼可见。

用量一目了然

本周（7 天）与 5 小时限额常驻菜单栏，百分比、进度条、重置时间一屏掌握，还有加油包余额。

Kimi 一键登录

采用 Kimi Code 官方接口，浏览器授权即登录；同时保留 API Key 方式，可在设置中自由切换。

更新自动探测

自动检测 Kimi Code CLI 新版本，系统通知 + 面板提醒，更新内容清晰可见，一键直达下载。

隐私安全

数据仅存储在本地，所有 API 只与 Kimi 官方通信，代码完全开源，可逐行审计。

Showcase

面板外观自动适配 macOS 深浅色模式，融入桌面，毫不突兀。

Auto Update

Kimi Code 更新，
不再错过

自动探测 Kimi Code CLI 新版本，从通知到面板，第一时间提醒你升级。

  • 系统级通知，新版本发布即刻推送
  • 更新日志直接展示，改了什么一目了然
  • 一键安装更新，也可以「稍后再说」

官方直连

开源可审计

MIT 协议全量开源，每一行代码都在 GitHub 上，欢迎审阅与共建。
